Vía un [tweet](http://twitter.com/minid/statuses/726556482) de [minid](http://www.minid.net/2008/02/18/dia-internacional-del-403-para-internet-explorer/) me acabo de enterar de que el día 4 de marzo se celebra el _Día internacional del 403 para Internet Explorer_, que propone mostrar un error 403 (acceso denegado) a todo el que visite tu web con el Internet Explorer. Hay un plugin ['Día403'](http://dizque.lacalabaza.net/sotanos/2008/02/dia-403/) para el WordPress y todo.

Yo, por supuesto, me adhiero a la propuesta rotundamente, pero [sin prohibir la entrada a nadie](http://www.kabytes.com/general/prohibir-el-acceso-de-visitantes-con-internet-explorer/), que los pobrecitos ya bastante tienen con usar el IE.

PD: Para _rebelarte_ todo el año (y de una manera más pacífica) recomiendo el plugin [NOIE](http://www.hnkweb.com/code/noie/) de [Hanok](http://www.hnkweb.com/)
