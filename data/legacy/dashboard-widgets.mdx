Los nuevos dashboard widgets son una gran idea que en las anteriores versiones del WordPress ya estaba implementada en forma de plugin (el magnífico [myDashboard](http://dev.clearskys.net/Wordpress/MyDashboard)) y que ahora, a partir de la 2.5 vienen _de serie_. A mi me resultan muy útiles, pues desde el dashboard puedes controlar todo de un vistazo. Además, como son personalizables casi todo lo que se te ocurra se puede transformar en un dashboard widget...

Estos son lo widgets que me parecen más útiles (que no se porque no están integrados en el _trunk_ del WordPress, la verdad)

- [Berri Technorati Reactions on Dashboard](http://www.berriart.com/technorati-reactions-dashboard-plugin/ "Visit plugin homepage") (porque, sinceramente, el Blog Search de Google apesta...)
- [Dashboard: Draft Posts](http://www.viper007bond.com/wordpress-plugins/dashboard-draft-posts/ "Visit plugin homepage")
- [Dashboard: Latest Spam](http://www.viper007bond.com/wordpress-plugins/dashboard-latest-spam/ "Visit plugin homepage")
- [Dashboard: Scheduled Posts](http://www.viper007bond.com/wordpress-plugins/dashboard-scheduled-posts/ "Visit plugin homepage")
- [WordPress.com Stats](http://wordpress.org/extend/plugins/stats/ "Visit plugin homepage") (no es un widget en si mismo, pero al usarlo en un WP2.5 pone uno en el dashboard)

Casi todos de [Viper007Bond](http://www.viper007bond.com/), desarrollador de WordPress, menuda máquina :)

Y no es un dashboard widgets, pero deberiaís tener instalado el [Dashboard Widget Manager](http://www.viper007bond.com/wordpress-plugins/dashboard-widget-manager/ "Visit plugin homepage") si o si: te permite organizar todos los dashboards widgets (ordenarlos, desactivarlos, etc...)

Non-Widget Bonus:

- [Admin Drop Down Menu](http://planetozh.com/blog/my-projects/wordpress-admin-menu-drop-down-css/ "Visit plugin homepage"), convierte la barra de menus en una lista desplegable. Otra maravilla que debería venir de serie en el WordPress.
- [Remove Max Width](http://dd32.id.au/wordpress-plugins/remove-max-width/ "Visit plugin homepage"), para eliminar el max-width de la zona de administración (si tienes un panorámico lo agradecerás)
- [Global Plugin Update Notice](http://www.viper007bond.com/wordpress-plugins/global-plugin-update-notice/ "Visit plugin homepage"), y evita tener que ir mirando plugin por plugin si hay actualizaciones...
