```php
// Rick - http://rick.jinabs.com/

require "Becker.inc";

//Initializing
$date = date("j, n, Y");
$world = array('earth', 'sky');
$alma = array();
$seeHer = 0;
$seeMe = 0;
$believeGod = FALSE;

while ($date) {
	foreach($world as $element) {
		$element.smile = TRUE;
	}
	array_push($alma, "Sol");
	$seeHer++;
	$seeHer++;
	$seeMe++;
	$believeGod = TRUE;
}
```

El original es este:

> Hoy la tierra y los cielos me sonríen, hoy llega al fondo de mi alma el sol, hoy la he visto..., la he visto y me ha mirado..., ¡hoy creo en Dios!
