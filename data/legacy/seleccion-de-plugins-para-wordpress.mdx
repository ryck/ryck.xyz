Esta es otra lista de esas de plugin indispensables para el Wordpress; o casi, porque indispensables la verdad es que no son, pero útiles un rato largo:

- [FeedBurner FeedSmith](http://www.feedburner.com/fb/a/help/wordpress_quickstart 'Visit plugin homepage'): Para redirigir tu feed automágicamente a FeedBurner. Te permite aligerar un poco tu servidor y llevar las estadísticas del feed...
- [Trackback Validator](http://trackback.cs.rice.edu/ 'Visit plugin homepage'): Elimina el spam de los trackbacks.
- [Akismet](http://akismet.com/ 'Visit plugin homepage'): No necesita presentación, es el anti-spam definitivo.
- [Code Snippet](http://blog.enargi.com/codesnippet/ 'Visit plugin homepage'): Para colorear todo el código fuente que muestres.
- [Exec-PHP](http://bluesome.net/post/2005/08/18/50/ 'Visit plugin homepage'): Permite la ejecución de PHP en tus entradas y páginas. Indispensable si quieres guarrear con tu blog.
- [Favatars](http://dev.wp-plugins.org/wiki/favatars 'Visit plugin homepage'): Es a los favicons lo que el gravatar a los avatars. Casí nadie lo tiene implementado para sus track/pingback, y es una pena, porque queda muy bien.
- [Google Analyticator](http://cavemonkey50.com/code/google-analyticator/ 'Visit plugin homepage'): Para usar las estadísticas del tito Google
- [Google XML Sitemaps](http://www.arnebrachhold.de/redir/sitemap-home/ 'Visit plugin homepage') : Para usar los sitemaps del tito Google
- [Gravatar](http://www.gravatar.com/implement.php#section_2_2 'Visit plugin homepage'): Gravatars para todo el mundo, que siempre queda resultón.
- [Lighter Admin Drop Menus](http://www.italyisfalling.com/lighter-admin-drop-menus-wordpress-plugin 'Visit plugin homepage'): El plugin definitivo para la parte de administración del WP. Solo modifica los menús para convertirlos en verticales desplegables. Tan bonito como eficiente. Y no se lleva mal con los demás plugins. Olvidaros de spotmilk y esas cosas, de verdad.
- [myComments](http://defmay.com/wordpress/plugins/mycomments 'Visit plugin homepage'): Otro plugin que casi nadie usa. Sirve para seguir el rastro de tus comentarios, en plan CoComent, pero de una manera mucho menos intrusiva y cómoda.
- [MyDashboard](http://dev.clearskys.net/Wordpress/MyDashboard 'Visit plugin homepage'): Otra joya. Debería incluirlo en el WP por defecto. Convierte el soso Dashboard por defecto en una página de inicio en plan iGoogle o Netvibes. Esta empezando, pero es muy robusto y ágil, además de tener un montón de posibilidades. No os olvideis de los gaggets de [gafeman](http://www.viciao2k3.net/wordpress/mi-dashboard/ 'Visit plugin homepage')
- [Optimal Title](http://elasticdog.com/2004/09/optimal-title/ 'Visit plugin homepage'): Cambiar el orden de los títulos de las entradas. Una chorrada a priori, pero queda muchísimo más claro para todo el mundo (incluidos los buscadores, claro). Otra de esas cosas que debería incluir el WP _per se_...
- [Pagebar](http://www.elektroelch.de/hacks/wp/pagebar.html 'Visit plugin homepage'): Cambia esos poco eficientes _anterior_ y _siguiente_ por una barrita con números ;)
- [Smilies Themer](http://rick.jinlabs.com/code/smilies-themer/ 'Visit plugin homepage'): Vale que te gusten los emoticonos, pero cambia los feotes que trae el WP por otros mas majos! Además, el creador es un tio majisimo! XD
- [Who Sees Ads ?](http://planetozh.com/blog/my-projects/wordpress-plugin-who-sees-ads-control-adsense-display/ 'Visit plugin homepage'): El plugin definitivo para controlar donde, cuando, como _y encima de quien_ salen tus anuncios.
- [WordPress Database Backup](http://www.ilfilosofo.com/blog/wp-db-backup 'Visit plugin homepage'): Eso de que te mande al mail cada cuanto tu quieras una copia de seguridad de tu BBDD no tiene precio.

Hay muchos más tan buenos como estos (ejem, los [míos](http://rick.jinlabs.com/code), ejem) pero tampoco vamos a pasarnos, ya sabéis que usar plugin porque si no es buena idea. E intentar que los que tengáis sean todo lo eficientes que puedan para no sobrecargar la carga del blog.
