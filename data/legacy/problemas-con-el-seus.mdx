Si al iniciar el SEUS (Sony Ericsson Update Service) te encuentras con este error:

> Failed checking application versions, please reinstall the application and make sure you have the latest version.

La solución es tan simple (y tan triste) como activar el tema Classic del Windows, actualizar el teléfono y volver a poner el tema de antes, claro está... Menuda pifia de error...
